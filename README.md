## 다함께 자격증 공부하자!
# 다자공

자격증 시험을 준비하는 사람들을 위한 통합 플랫폼

---


## 팀원 소개

- **박진희** (팀장) - 책 리뷰 페이지 / 관리자 페이지
- **김은호** - 로그인 / 회원가입 / 마이페이지
- **한수열** - QnA 페이지



---

## 개발 환경

### 개발 도구
- **IDE**: Spring Tools for Eclipse (STS)
- **버전관리**: Git / GitHub

### 백엔드
- **언어**: Java (JDK 21)
- **프레임워크**: Spring Boot 4.0.5
- **ORM**: MyBatis
- **서버**: Apache Tomcat 11.0

### 프론트엔드
- Thymeleaf
- jQuery, Bootstrap, AJAX

### 데이터베이스
- Oracle
